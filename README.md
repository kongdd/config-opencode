# opencode config

```powershell
git clone https://github.com/kongdd/config-opencode.git $HOME\.config\opencode
```

## 登录授权
```powershell
npm install -g opencode-ai
opencode auth login
```

## References

- [opencode-antigravity-auth](https://github.com/NoeFabris/opencode-antigravity-auth): OpenCode 插件，支持通过 Google OAuth 访问 Antigravity 模型与配额。

- [vibe-kanban](https://github.com/BloopAI/vibe-kanban): AI 编码代理管理看板，支持多代理编排、任务跟踪与远程开发。

- [Antigravity-Manager](https://github.com/lbjlaq/Antigravity-Manager): 桌面端 Antigravity 账号管理与协议中转工具，支持配额监控与模型路由。

- [portal](https://github.com/hosenur/portal): OpenCode Web UI，支持移动端访问、会话管理与内置终端。

- [worktrunk](https://github.com/max-sixty/worktrunk): Git worktree 管理 CLI，优化了 AI 代理的并行开发工作流。

- [oh-my-opencode](https://github.com/code-yeongyu/oh-my-opencode): OpenCode 插件，提供多模型编排、Claude Code 兼容层及 Sisyphus 代理增强。
